# AI Prompt Engineering Workshop

A modern, interactive HTML presentation for teaching prompt engineering to beginners, originally created for a 60 to 90 minute adult education class focused on older adult learners.

The entire presentation is contained in a single HTML file with no external dependencies. It works completely offline and is easy to customize for your own classes, workshops, libraries, or community events.

## Features

- Single HTML file
- Pure HTML, CSS, and JavaScript
- No frameworks
- No external libraries
- Works completely offline
- Responsive design
- Keyboard navigation
- Fullscreen presentation mode
- Animated slide transitions
- Progress bar and slide counter
- Speaker notes
- Clickable table of contents
- Interactive prompt examples
- Expandable tips
- Accessibility focused design
- Large typography for classroom presentations

## Topics Covered

- Why prompt engineering matters
- Anatomy of a great prompt
- Weak vs. strong prompts
- Real world prompting examples
- Prompt engineering techniques
- Common prompting mistakes
- Prompt templates
- Hands on classroom exercises
- Using AI as a collaborator instead of a search engine

## Audience

This presentation was designed for learners who already have a basic understanding of AI and Large Language Models, including:

- What LLMs are
- Tokens
- Transformer architecture
- AI model history
- Basic prompting

## Running the Presentation

Simply open the HTML file in any modern web browser.

No installation, build tools, or internet connection is required.

## Customization

Everything is contained in one file, making it easy to edit.

- Exercises
- Animations
- Navigation
